# v2网易云移动端

## 下载项目
```
git clone https://github.com/zxcvbnm456821379/v2musicmobile.git
```

### 运行项目
```
npm run serve
```

### 打包项目
```
npm run build
```

# 目前实现了首页渲染（还有一处bug没改），热歌榜和搜索模块还在开发中。。。。
> 技术栈 ： v2、axios、vue-router 纯练手项目没有用ui组件库.
